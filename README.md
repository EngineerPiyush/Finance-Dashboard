# Finance Dashboard

This is a simple finance dashboard built using React and Tailwind CSS.  
The goal of this project is to display financial data in a clean and easy to understand way.

## Features

- View total balance, income and expenses
- View all transactions with date, category, amount and type
- Search transactions by category
- Filter transactions (income / expense)
- Role-based UI (Viewer / Admin)
- Insights like total income, total expense and top spending category
- Pie chart for category-wise spending
- Line chart for monthly balance trend

## Tech Used

- React (Vite)
- Tailwind CSS
- Recharts (for charts)

## How to Run

1. Clone the repository
2. Go to project folder
cd finance-dashboard
Install dependencies
npm install
Run the project
npm run dev
